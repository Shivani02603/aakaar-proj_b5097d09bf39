**Primary Users:** Users who upload Excel files and ask natural-language questions about the data  
**Tech Constraints:** Python FastAPI (backend), Next.js (frontend), PostgreSQL with pgvector extension, Docker Compose deployment, OpenAI text-embedding-3-small, Google Generative AI SDK (gemini-2.0-flash), pandas

**Project Summary:** This project builds a web application that allows users to upload Excel (.xlsx) files and ask natural-language questions about the data. The system ingests files by parsing them with pandas, chunking the content, embedding chunks with OpenAI, and storing vectors in pgvector. On query, it retrieves relevant chunks via cosine similarity and generates answers with source citations using Google's Gemini model, all within named chat sessions stored in PostgreSQL.
